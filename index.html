Animation blueprint
Hero Text Reveal GSAP
SplitText chia từng ký tự. Mỗi char: y:60, opacity:0 → y:0, opacity:1, stagger: 0.03s, ease: power3.out. Headline trước, sub-copy sau 0.4s.
Parallax Hero Image GSAP ScrollTrigger
Hero image: yPercent: -15 trên scrub:1. Product cards nền: yPercent: -8 tốc độ chậm hơn. Dùng will-change: transform.
Counter Count-Up GSAP + Observer
IntersectionObserver: once:true. GSAP: {val:0} → {val:1500, duration:2, ease:"power1.out", onUpdate}. Icon pop: scale(0,0.8) → scale(1) với stagger 0.15s.
Tab Slide-in CSS
Outgoing: opacity:1 → 0, translateX(0 → -20px). Incoming: opacity:0 → 1, translateX(30px → 0). Duration: 0.28s, ease: ease-out. Class toggle.
Roadmap Line Draw GSAP ScrollTrigger
SVG path với stroke-dasharray: totalLength. ScrollTrigger scrub:true animates strokeDashoffset: length → 0. Mỗi step khi vào viewport: box-shadow: 0 0 14px #FF8C00.
Magnetic Button Vanilla JS
Detect khoảng cách cursor → center btn. Nếu < 100px: translate(dx×0.3, dy×0.3), transition 0.15s. onMouseLeave: reset về 0,0. Ease: spring.
Social Icon Spin CSS
.social-icon:hover { transform: rotate(360deg); transition: transform 0.5s cubic-bezier(.4,0,.2,1); color: #FF8C00; }. Duration 0.5s đủ nhanh, không chóng mặt.

Breakpoint strategy
📱
Mobile
< 768px
· Hero: 1 cột, text trên
· Nav: Hamburger → drawer
· Counters: 2×2 grid
· Products: 1 cột scroll
· Roadmap: vertical
· Swipe gesture tabs
💻
Tablet
768–1024px
· Hero: 2 cột compact
· Nav: full width
· Counters: 4 cột
· Products: 2 cột
· Roadmap: horizontal
· Support: tab + icon
🖥️
Desktop
> 1024px
· Hero: 2 cột full
· Nav: full + mega hint
· Counters: 4 cột lớn
· Products: 3 cột
· Roadmap: full motion
· Magnetic button active
Mobile-specific UX
Tabs (S5) dùng touch swipe trái/phải với HammerJS hoặc pointer events native. Product grid dùng horizontal scroll snap thay hover (tap để toggle overlay). Roadmap collapse về dạng accordion. Magnetic button disabled trên touch devices.
